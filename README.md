### Setup Instructions
1. Clone project
2. Run npm install in project directory
3. Run npm start
4. Go to http://localhost:8080/


### Application Instructions

Click window to start. 
Use mouse to look around.
WASD + Spacebar and Shift to translate camera in all six directions.
